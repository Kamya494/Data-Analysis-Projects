# Stock Market Analysis using NumPy

This project analyzes a synthetic stock market dataset using NumPy. It focuses on practicing array manipulation, statistical analysis, sorting, indexing, and filtering.

## Dataset
- 500 stock market records
- Columns:
  - Open
  - High
  - Low
  - Close
  - Volume

## Analysis Performed
- Descriptive statistics
- Price range analysis
- Daily price movement
- Percentile analysis
- Top 10 closing prices
- Volume analysis
- Boolean indexing
- Sorting
- Top 10 volume

## Tools Used
- Python
- NumPy
- Pandas
