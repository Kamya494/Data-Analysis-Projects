# Student Performance Analysis

## Overview

This project analyzes student performance on a synthetic dataset using **NumPy** and **Pandas**. It focuses on applying core NumPy concepts to solve real data analysis problems.

## Concepts Covered

- NumPy arrays
- Indexing and slicing
- Statistical analysis
- Boolean filtering
- `np.where()`
- `np.any()` and `np.all()`
- `np.argmax()` and `np.argmin()`
- `np.argsort()`
- Student ranking

## Files

- `numpy_DA_project.ipynb` – Jupyter Notebook
- `student_performance.csv` – Dataset

## Tools Used

- Python
- NumPy
- Pandas
