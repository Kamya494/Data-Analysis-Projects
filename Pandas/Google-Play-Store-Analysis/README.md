# Google Play Store Analysis

A data analysis project exploring the Google Play Store dataset using **Python, Pandas, and Matplotlib**.

**Dataset:** Google Play Store Apps Dataset (Source: Kaggle)

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization

## Analysis Includes
- Top app categories by installs.
- Number of apps in each category.
- Free vs Paid apps.
- Distribution of app ratings.
- Content rating analysis.

## Tools Used
- Python
- Pandas
- Matplotlib
