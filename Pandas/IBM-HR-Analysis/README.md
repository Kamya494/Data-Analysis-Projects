# HR Data Analytics

## Overview

This project performs Exploratory Data Analysis (EDA) on the IBM HR Analytics Employee Attrition & Performance dataset using Python.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed

- Data Cleaning
- Descriptive Statistics
- Employee Distribution
- Age Analysis
- Gender Analysis
- Department Analysis
- Attrition Analysis
- Job Satisfaction Analysis
- Monthly Income Analysis
- Salary Hike Analysis
- Work-Life Balance Analysis

## Key Insights

- Research & Development has the highest number of employees.
- Most employees are between 30–40 years old.
- Higher job satisfaction is associated with lower attrition.
- Employees working overtime are more likely to leave the company.
- Salary hikes are fairly consistent across different job roles.

## Dataset

IBM HR Analytics Employee Attrition & Performance Dataset (Source : Kaggle)
