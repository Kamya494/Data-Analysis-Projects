# Netflix Data Analysis

This project explores the Netflix Movies and TV Shows dataset using Python, Pandas, NumPy, and Matplotlib.

## Project Overview
The dataset was cleaned, analyzed, and visualized to identify trends in Netflix content.

## Analysis Performed
- Movies vs TV Shows
- Top 10 Countries
- Top Genres
- Content Ratings
- Release Year Distribution
- Content Added by Year

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
Netflix Movies and TV Shows dataset (Source : Kaggle)
