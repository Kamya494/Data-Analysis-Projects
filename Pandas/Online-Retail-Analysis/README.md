# Online Retail Analysis

## Overview

This project analyses Online Retail dataset, taked from the UCI Machine Learning Repository, using **Pandas**. 
It focuses on applying core Pandas concepts to solve real data analysis problems, as well as answer real business questions.

## Project Objectives
- Clean missing and duplicate data
- Analyse top-selling products
- Calculate monthly revenue
- Explore sales by country
- Generate business insights

## Files

- `Online_Retail-2` – Jupyter Notebook
- `Online Retail 2.xlsx` – Dataset

## Tools Used

- Python
- Pandas
